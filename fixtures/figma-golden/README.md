# Golden Figma фикстуры

Эта папка содержит репрезентативные экспорты Figma, используемые для регрессионного тестирования.

## Назначение

- Проверка ingest и нормализации
- Проверка генерации адаптивных правил
- Сравнение результатов экспортеров во времени

## Текущие фикстуры

- `mobile-login-screen.json`: базовый экран с auto-layout и смесью text/image узлов
