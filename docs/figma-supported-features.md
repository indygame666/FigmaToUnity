# Поддерживаемые возможности Figma (MVP)

Этот документ фиксирует область поддерживаемого импорта для первой production-ready версии конвертера Figma -> Unity.

## Полностью поддерживается

- Frame и вложенная иерархия frame
- Текстовые слои (семейство шрифта, размер, насыщенность, межстрочный интервал, выравнивание)
- Узлы rectangle/image с экспортом графических ассетов
- Базовый Auto Layout (горизонтальный/вертикальный поток, отступы, расстояния)
- Базовые Constraints (left/right/top/bottom + stretch)
- Цвета заливки и обводки
- Переиспользуемые компоненты (instances с привязкой по стабильному component ID)

## Поддерживается частично

- Варианты компонентов (одна ось вариантов + fallback по умолчанию)
- Радиусы углов по каждому углу
- Прозрачность и режимы наложения (с безопасным fallback, где нет прямого соответствия)
- Эффекты (в MVP только тень с уровнями предупреждений)

## Не поддерживается в MVP

- Сложные векторные пути и булевы операции
- Многомерная логика вариантов с условным runtime-переключением
- Расширенные текстовые эффекты и OpenType-возможности
- Интерактивные прототипы Figma (навигация/анимации)

## Уровни предупреждений

- `info`: узел сконвертирован с эквивалентным поведением Unity
- `warning`: узел сконвертирован с приближением/fallback
- `error`: узел пропущен из-за неподдерживаемой возможности

## Критерии готовности

- >=90% golden-экранов импортируются без `error`
- Все неподдерживаемые возможности отражаются в диагностике с рекомендациями
