# Scorecard выбора Unity-таргета

Используйте этот scorecard после двух спринтов разработки, чтобы выбрать основной таргет экспорта.

## Входные показатели (0..100)

- Покрытие Auto Layout
- Покрытие Constraints
- Покрытие эффектов
- Оценка runtime-производительности
- Оценка удобства донастройки в Unity Editor

## Правило принятия решения

- `totalScore >= 90`: кандидат в основной таргет
- Иначе остается вторичным/экспериментальным

Вспомогательная логика расчета находится в `backend/src/exporters/scorecard.ts`.
