# Метрики качества

Отслеживайте эти KPI во время стабилизации и rollout:

- Доля успешных golden-импортов (цель >= 90%)
- Среднее время импорта одного экрана (цель <= 10 секунд)
- Доля ручных правок после импорта (цель <= 20%)
- Доля сессий импорта без падений (цель >= 99%)

## Источники сбора

- Результаты smoke-проверок на golden-экранах из `fixtures/figma-golden`
- Сценарий `Preview -> Import -> Reimport` на реальных экранах
- Отчеты импорта, сформированные в окне редактора
